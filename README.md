#  Shopping Cart Billing System (Python)

##  Overview
A simple and user-friendly **console-based billing system** made with Python.  
Users can browse products, add items to a cart, view the cart, and checkout with an auto-generated bill.

This system includes **smart product matching**, so inputs like:
- `rice`
- `RICE`
- `Rice 1KG`
- `rice1kg`

will automatically match **Rice 1kg**.

---

##  Features
- ✔️ Product catalog with price & available stock  
- ✔️ Smart flexible name matching  
- ✔️ Add items to cart  
- ✔️ Stock auto-update  
- ✔️ View cart with totals  
- ✔️ Checkout with final bill  
- ✔️ Clean and human-readable code  
- ✔️ No external libraries required  

---

 ## Technologies / Tools Used

- Python 3

- Dictionaries & Lists for data storage

- Functions for modular design

- String processing for product-name normalization

- Loops & conditions for program flow

- Terminal/Console for user interaction

- VS Code for writing & running the program
